# Xonix Console

Консольная реализация классической аркадной игры Xonix на PureBasic 6.x для Windows.

## Цель игры

Игрок перемещается по закрашенной территории и захватывает незакрашенное поле,
проводя через него линии. Когда линия возвращается на закрашенную территорию,
она замыкается, а область, в которой нет шариков, становится захваченной.

Для перехода на следующий уровень по умолчанию необходимо закрасить не менее
75% поля. Столкновение с врагом или собственным незавершённым следом отнимает
одну жизнь.

## Враги

В игре есть два типа противников:

- `●` — шарик. Движется внутри незакрашенной области, отражается от её границ и
  уничтожает игрока при столкновении с ним или с незавершённым следом.
- `X` — охотник. Движется только по закрашенной территории и отражается от её
  границ. Столкновение охотника с игроком отнимает жизнь.

На первом уровне появляется один охотник. При переходе на каждый следующий
уровень количество шариков и охотников увеличивается на один, а скорость
шариков возрастает. После потери жизни охотники возвращаются на верхнюю границу
поля.

## Управление

| Клавиша | Действие |
|---|---|
| `W/A/S/D` или стрелки | Изменить направление движения игрока |
| `P` | Пауза или продолжение игры |
| `F1` | Показать или скрыть справку |
| `+` / `-` | Увеличить или уменьшить скорость шариков |
| `N` | Добавить ещё один шарик |
| `H` | Добавить ещё одного охотника |
| `Esc` или `Q` | Открыть подтверждение выхода |
| `Y` / `N` | Подтвердить или отменить выход |
| `R` | Начать заново после окончания игры |

Добавленные вручную противники учитываются при переходе на следующий уровень.

## HUD

Нижняя строка игрового экрана показывает:

- `Lvl` — текущий уровень;
- `Lives` — количество оставшихся жизней;
- `Fill` — процент закрашенного поля;
- `Goal` — процент, необходимый для завершения уровня;
- `En` — количество шариков;
- `Hun` — количество охотников;
- `Spd` — текущую скорость шариков.

## Конфигурация

При запуске игра читает файл `xonix.ini` из текущего рабочего каталога, а при
выходе сохраняет в него актуальные значения.

| Параметр | Значение по умолчанию | Назначение |
|---|---:|---|
| `TargetPercent` | `75` | Процент поля для завершения уровня |
| `StartLives` | `3` | Начальное количество жизней |
| `StartEnemies` | `2` | Начальное количество шариков |
| `MaxFPS` | `30` | Максимальная частота обновления экрана |
| `BaseSpeed` | `1.2` | Начальная скорость шариков |
| `SpeedGrowth` | `0.15` | Прирост скорости на новом уровне |
| `SpeedChange` | `0.1` | Шаг изменения скорости клавишами `+` и `-` |

Начальное количество охотников равно одному и задаётся константой
`#DEFAULT_START_HUNTERS` в исходном коде. Интервал их движения задаётся
константой `#HUNTER_STEP_MS`.

## Запуск

Для запуска готовой версии откройте `xonix_console.exe`. Игра использует размер
текущего окна консоли; минимальный размер игрового поля — 40 × 20 символов.

Требуется Windows с поддержкой Unicode в консоли.

## Сборка

Установите PureBasic 6.x и выполните из каталога проекта:

```powershell
& 'C:\Program Files\PureBasic\Compilers\pbcompiler.exe' `
  '.\xonix_console.pb' /EXE '.\xonix_console.exe' `
  /ICON '.\Xonix.ico' /CONSOLE
```

Основной исходный файл — `xonix_console.pb`. Иконка `Xonix.ico` встраивается в
исполняемый файл во время сборки.
